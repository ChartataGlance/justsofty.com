# JustSofty Static Website

Static website for JustSofty consulting services.

## Local preview

Open `index.html` in a browser, or run:

```bash
python3 -m http.server 8080
```

Then visit:

```text
http://127.0.0.1:8080
```

## GitHub Pages

This site is static and ready for GitHub Pages. Push the repo and enable Pages from the repository settings.
